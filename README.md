# Siddhanth Nandan Seshadri - Biography Website

A Wikipedia-style biography website celebrating Siddhanth Nandan Seshadri.

## About

This website presents a comprehensive biographical overview of Siddhanth Nandan Seshadri, featuring:

- **Professional Layout**: Wikipedia-inspired design with infobox and navigation
- **Biography Section**: Overview of his life and character
- **Interests & Talents**: Highlights of his diverse skills and passions
- **Accomplishments**: Notable achievements and milestones
- **Personal Life**: Insights into his values and relationships
- **Legacy**: His impact and influence

## Files

- `index.html` - Main biography page with full content
- `style.css` - Professional styling inspired by Wikipedia
- `README.md` - This file

## Features

✨ Responsive design that works on desktop and mobile
✨ Clean, professional Wikipedia-style layout
✨ Organized sections with table of contents
✨ Infobox with key information
✨ Smooth scrolling navigation
✨ Modern color scheme with gradient header

## How to View

1. Open `index.html` in your web browser
2. Or deploy to GitHub Pages for online hosting

To enable GitHub Pages:
1. Go to Settings > Pages
2. Select `main` branch as source
3. Your site will be available at `https://lawrence-w17.github.io/Sid`
